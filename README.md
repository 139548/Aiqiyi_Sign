## **Aiqiyi-Sign**


### 脚本功能：

​	爱奇艺会员签到打卡，增加经验值，增加爱奇艺会员天数（连续打卡28天，可以增加10天会员）

### 特点：

​	不需要手动获取cookie值，可以实现手机号+密码登录，自动获取cookie实现签到

### 使用方式：

​	直接下载action.py这个文件，在终端运行下面这个命令

```shell
python .\action.py 电话号码 密码
```
    
### 注意事项：

​    需要是爱奇艺会员

### 建议：

    关注爱奇艺的公众号，能接受到会员天数的增加信息

### 补充：

​	后期会加推送等
